Very rudimentary (somewhat on par with Super Mario Bros. sound engine's features) NES music engine.

See [mush.rb](/mush.rb).

Music credits:

- James Montagna (melody/motif composition)
- Madeline Lim (MIDI arrangements made with [Auxy](http://auxy.co))

Includes a modified [c64asm](https://github.com/drbig/c64asm) used for as a Ruby DSL for generating 6502 code.
